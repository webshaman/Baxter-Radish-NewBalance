Required ranking methodology

For each artist, we should:

  1. Compile every track by that artist where Tyler Spry is credited as producer, co-producer, writer, composer, engineer, or programmer.
  2. Prefer producer/co-producer credits when the slide claim is “producer.”
  3. Pull Spotify stream counts for the verified tracks.
  4. Select the highest-streaming verified Tyler Spry credit for that artist.
  5. Use the official artist / VEVO / label YouTube video for the modal.

Spotify stream-count caveat

Spotify’s public Web API does not expose all-time stream counts. Counts generally need to be captured from:

  • Spotify desktop/web UI where visible,
  • Spotify for Artists / label-side access,
  • Chartmetric, Soundcharts, Songstats, Kworb, or similar data sources,
  • or manually documented screenshots.

Until one of those sources is used, any “biggest song” assessment is incomplete.
